# Reglas de Intervención (AI Memory)

Estas reglas deben ser seguidas por cualquier asistente de IA que trabaje en este repositorio:

1. **Intervención Mínima**: Realizar solo los cambios estrictamente necesarios para cumplir con el objetivo solicitado. No refactorizar o cambiar estilos sin autorización explícita.
2. **Cambios No Solicitados**: NO realizar cambios que no hayan sido pedidos por el usuario.
3. **Dudas y Clarificación**: Si hay alguna ambigüedad o duda en la tarea, se DEBE preguntar al usuario antes de proceder con el código.
4. **Mejoras e Ideas**: Si tienes sugerencias de mejora (UX, rendimiento, seguridad, etc.), debes explicar el cambio propuesto y solicitar aprobación ANTES de implementarlo.
5. **Preservar Estilo**: Mantener la consistencia con el diseño "Sagason 4K" (glassmorphism, tipografía Bebas Neue/Space Grotesk) y los patrones de código existentes.
